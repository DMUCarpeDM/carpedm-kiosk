# 고령층 UI 최적화 체크리스트 (근거 기반)
### CarpeDM 키오스크 — 디자인 검수 + 발표 근거용

> 출처: WCAG 2.2 (1.4.3 대비 / 1.4.4 텍스트 확대 / 1.4.11 비텍스트 대비), Apple HIG·Google Material(터치 최소치), 고령층 UX 문헌(Toptal 2023, Netguru 2026, TPGi 2025, eSEOspace 2025 등), KS X 9211(키오스크 접근성), 문체부·한글문화연대 외래어 인식조사(2020).
> 사용법: 시안·구현 검수 시 각 항목 ✓. "근거" 열은 심사 Q&A 방어용. ⚠는 실측 필요.

## A. 시각 — 글자
| 항목 | 기준 | 근거 | ✓ |
|---|---|---|---|
| 본문 최소 크기 | 키오스크(50~70cm)에서 본문 ≥32px, 최소 20px 이하 금지 | 문헌 공통 16px는 모바일 근거리 기준 → 원거리 키오스크는 상향 필요 | ☐ |
| 글자 굵기 | 700 기본, 얇은 굵기 금지 | 얇은 폰트는 배경에 묻혀 고령 시인성 저하 | ☐ |
| 서체 | 산세리프, 장식체 금지. KoddiUD 온고딕 | 화면 가독성·UD 검증 서체 | ☐ |
| 행간 | 1.4~1.5 | 충분한 줄간격이 읽기 편의 | ☐ |
| 확대 기능 | [글자 크게] ×1.25, 레이아웃 깨짐 없이 | WCAG 1.4.4 텍스트 200% 확대 / 사용자 제어 원칙 | ☐ |

## B. 시각 — 색·대비
| 항목 | 기준 | 근거 | ✓ |
|---|---|---|---|
| 본문 대비 | ≥7:1(AAA 목표), 최소 4.5:1 | 노화로 대비 민감도 저하, AA는 최소선 ⚠WebAIM 실측 | ☐ |
| 파랑 회피 | 주요 정보·구분에 파랑 단독 사용 금지 | 고령에게 파란색이 바래 보여 대비 저하 | ☐ |
| 색 단독 금지 | 정보는 색+글자+아이콘 3중 | 색각·대비 저하 대응(WCAG 1.4.1) | ☐ |
| 비텍스트 대비 | 버튼·아이콘 외곽도 ≥3:1 | WCAG 1.4.11 | ☐ |
| 흑백 테스트 | 회색조 변환에서도 식별 가능 | 색 의존 제거 검증 | ☐ |
