## B. 외래어 장벽 — **우리 차별화의 핵심 근거**

> 카페·패스트푸드 메뉴는 외래어(아메리카노·라떼·프라푸치노…)투성이다. 고령층에게 이는 사실상 외국어다.

| 사실 | 수치 | 출처 / 신뢰도 |
|---|---|---|
| 외국어 표현 이해도 (전 국민 평균) | 100점 만점 61.8점 | 문체부·한글문화연대 2020 / 높음 |
| 60% 이상이 이해하는 외국어 표현 | 3,500개 중 1,080개(30.8%) | 문체부·한글문화연대 2020 / 높음 |
| 60대 이하가 이해(60%+) | 1,378개(39.4%) | 문체부·한글문화연대 2020 / 높음 |
| **70세 이상이 이해(60%+)** | **242개(6.9%)** | 문체부·한글문화연대 2020 / 높음 |

**조사 개요**: 문체부 + 한글문화연대, 2020년 1~2월, 전국 16개 지역 14~79세 11,074명, 정부·언론에 쓰인 외국어 표현 3,500개 이해도 측정(70대는 개별면접).

> 💡 **해석**: 70세 이상은 외래어 표현의 **6.9%만** 60% 이상 이해한다. 즉 메뉴판의 영어 이름은 고령층에게 거의 통하지 않는다. → "메뉴명을 정확히 말해야 하는" 기존 음성주문은 고령층에 부적합하고, **우리말 표현("검은 탄산 물")을 의미로 해석하는** 우리 방식이 정당화된다.

**현장 정황(언론)**: 영어 메뉴판 앞에서 고령층이 "좋아하지도 않는 아메리카노만" 시키거나 "따뜻한 커피"로 회피 주문하는 사례 다수 (머니투데이 2021, 중앙일보 2021). QR코드·팝업창·모바일앱 같은 IT 용어도 세대 간 이해 격차가 큼. **(신뢰도: 중간 — 언론 취재 일화)**

## C. 단계별 사용 실패 — 페인포인트 구체화 (현장 일화)

> 통계(64.2% 등)를 "어디서 막히는지"로 구체화. 시험 후 구로구 현장 관찰로 검증·보강.

- **결제 흐름 혼란**: '결제'를 눌렀는데 메뉴를 다시 정하라는 안내가 뜨고, '취소'를 눌렀더니 주문 전체가 리셋됨 (월간중앙 2024). → *되돌리기 설계 + 단계 명확화 필요.*
- **판독 불가**: "눈이 침침한데 글씨가 작아 뭐라 써 있는지 모르겠다" (72세). → *큰 글씨·고대비.*
- **기억 부담**: "알려줘도 기억이 안 난다" (77세). → *학습 부담 제거 = 음성 주문.*
- **시간 압박**: 뒷사람 눈치(B장 통계 49%)로 서두르다 실수. → *타임아웃 없음.*

**(신뢰도: 중간 — 언론 인터뷰)**

| 사실 | 함의 | 출처 / 신뢰도 |
|---|---|---|

## E. 차별화 근거 — 기존 VUI/키오스크의 한계

| 사실 | 함의 | 출처 / 신뢰도 |
|---|---|---|
| 전통적 VUI는 **정형(directed) 대화**(예/아니오, 제한된 응답)에 적합하고, 자연어 발화 처리에는 경직됨 | 기존 음성주문은 정확한 명령을 요구 → 고령층 비정형 표현에 취약 | VUI 설계 특허 문헌(USPTO US10656908 등) / 높음(기술 문헌) |
| 기존 배리어프리 키오스크 음성 = TTS 화면 읽기 + 시각장애인 키패드 + 정형 음성주문 | 출력·접근성 중심. **고령층의 '의미 이해형' 입력은 공백** | 코레일유통·하나시스·NICE인프라 보도 / 높음 |

> 💡 **결론**: "정확한 메뉴명을 말해야 하는" 기존 방식의 한계 + 고령층 외래어 6.9% 장벽 → **LLM 기반 의미 이해형 음성 주문**이 우리만의 빈 공간.

## I. 고령층 표현 사전 — 외래어 매핑 강화 (가설)

> ⚠ **전부 가설**이다(신뢰도: 낮음 — 미검증). 시험 후 구로구 어르신 인터뷰로 실제 표현을 수집·검증·교체한다. B장(외래어 6.9%)이 이 사전의 존재 이유다.

### 양방향 매핑 (외래어 메뉴 ↔ 어르신 표현)
| 메뉴(외래어) | 어르신이 쓸 법한 우리말 표현(가설) |
|---|---|
| 아메리카노 | 진한 커피 / 쓴 커피 / 그냥 커피 / 검은 물 |
| 카페라떼 | 우유 든 커피 / 부드러운 커피 / 흰 커피 |
| 아이스 ○○ | 시원한 거 / 찬 거 / 얼음 든 거 |
| 핫(따뜻한) ○○ | 뜨신 거 / 뜨거운 거 / 따뜻한 거 |
| 콜라 | 검은 탄산 물 / 까만 사이다 |
| 사이다 | 투명한 탄산 / 쏘는 물 |
| 스무디·프라푸치노 | 갈아 만든 거 / 슬러시 같은 거 |
| 디저트(케이크·카스텔라) | 부드러운 거 / 달달한 빵 / 말랑한 거 |
| 베이글·크로플 | 빵 종류(설명 필요) |

### 속성 표현 (옵션 매핑, 가설)
- "안 매운 거 / 순한 거" → 맵기 낮춤
- "달달한 거 / 단 거" → 단맛·시럽 추가
- "안 단 거" → 시럽 제외
- "조금만 / 작은 거" → 사이즈 S

### 현장 수집 질문(시험 후 사용)
1. "○○ 드시고 싶으면 키오스크에 뭐라고 말씀하실 거예요?"
2. (메뉴 사진 제시) "이건 뭐라고 부르세요?"
3. 실제 주문 상황 재현 녹음(사전 동의).
